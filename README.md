## Quick Start

[Create a New Website](https://dotnet.github.io/docfx/#publish-to-github-pages)

Xây dựng trang web tài liệu kỹ thuật của bạn với docfx. Chuyển đổi tập hợp .NET, nhận xét mã XML, tệp Swagger API REST và đánh dấu thành các trang HTML được hiển thị, mô hình JSON hoặc tệp PDF.

### Create a New Website

```bash
dotnet tool update -g docfx

# To create a new docset, run:
docfx init

# To build the docset
docfx docfx.json --serve

# To preview your local changes, save changes
docfx docfx.json
```
